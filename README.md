# Health-Tracker
The Health Tracker Application is a Python-based console program developed using Object-Oriented Programming (OOP) principles to promote structured and maintainable code design. The application allows users to create and store a personal health profile, record daily body temperature readings, and monitor health trends over time through graphical analysis using Matplotlib.
The system validates user inputs such as age, gender, and date format to ensure accuracy and prevent errors. Temperature records are stored in CSV format for easy data management and retrieval. If a recorded temperature exceeds the normal range (above 37°C), the application provides a health warning.
Additionally, the program integrates a real-time COVID-19 API to display country-specific statistics, including total cases, deaths, recoveries, active cases, and critical cases. This feature demonstrates API integration using the Requests library.
Overall, the project showcases practical implementation of OOP concepts, file handling, input validation, API usage, and data visualization, making it a comprehensive and well-structured computer science project suitable for academic evaluation.
